#### 3 正则取url变量

```javascript
function parse_url(_url){ //定义函数
　　var pattern = /(\w+)=(\w+)/ig;//定义正则表达式
　　var parames = {};//定义数组
　　url.replace(pattern, function(a, b, c){
　　　　parames[b] = c;
　　});
　　return parames;//返回这个数组.
}
```
